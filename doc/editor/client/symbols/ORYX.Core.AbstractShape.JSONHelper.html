<static> ORYX.Core.AbstractShape.JSONHelper.eachChild(iterator, deep, modify)
Iterates over each child shape. 
// Increases the lowerRight x value of each direct child shape by one. 
myShapeAsJson.eachChild(function(shape, parentShape){
    shape.bounds.lowerRight.x = shape.bounds.lowerRight.x + 1;
    return shape;
}, false, true);
Parameters:
{Object} iterator
Iterator function getting a child shape and his parent as arguments.
{boolean} deep Optional, Default: false
Iterate recursively (childShapes of childShapes)
{boolean} modify Optional, Default: false
If true, the result of the iterator function is taken as new shape, return false to delete it. This enables modifying the object while iterating through the child shapes.
